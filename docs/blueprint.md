# **App Name**: DramaVerse Admin

## Core Features:

- Series Browsing (User View): Users can view a catalog of available drama series, navigate through genres, and search for specific titles.
- Series Details Page: A dedicated page for each series, displaying details such as description, cast, seasons/episodes, and cover art.
- Admin Panel Login: Secure login for administrators with 'admin' as both username and password, protecting the content management features.
- Series Management (Admin): Administrators can add new series, and edit or delete existing series information including title, description, and images.
- AI-powered Series Description Assistant: A tool for admins to automatically generate or refine series descriptions based on a few keywords or plot points, streamlining content entry.
- Image Upload and Management: Functionality within the admin panel to upload and link cover art and other images to series and episodes.

## Style Guidelines:

- Color Scheme: Dark. Inspired by cinematic evenings and premium entertainment, a dark aesthetic provides a sophisticated backdrop for vibrant content.
- Primary Color: #6629CC. A deep, rich indigo-violet, conveying sophistication and a premium feel, used for primary elements and interactive components.
- Background Color: #17151A. A very dark, almost-black base with a subtle indigo undertone, providing excellent contrast for text and visual content.
- Accent Color: #4DB2FF. A bright, clear sky-blue for highlights, call-to-action buttons, and subtle visual cues, creating a dynamic contrast with the darker palette.
- Main font: 'Inter', a grotesque sans-serif for a modern, objective, and highly legible appearance suitable for both headlines and body text.
- Use minimalist, outline-style icons to maintain a sleek and modern visual language consistent with a premium streaming service aesthetic.
- Employ a responsive grid layout for series listings, ensuring an optimal viewing experience across various devices, with clear hierarchy for series details.
- Subtle and smooth transitions for navigation elements and interactive components, enhancing the user experience without being distracting.
